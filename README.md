# FairPay

This project is built with React Native and uses Expo for development and deployment.

## Prerequisites

Before you start, make sure you have the following software installed:

- Node.js 18 or higher.
- Expo CLI latest (you can install it globally by running `npm install -g expo-cli`).

## Installation
Run the following command to install the project dependencies:

   ```bash
   npm install
   ```

## Execution
Run the following command to start development server:

   ```bash
   npm start
   ```

Feel free to use Android or IOS emulators or your phisical devices using Expo Go app.  Also you can test the app in your web browser by pressing w in the console.

If you are running backend on local in `config.js` you can replace the host by localhost.
